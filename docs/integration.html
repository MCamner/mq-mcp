mq-mcp Integration Map

mq-mcp connects local MCP tools with repo-aware workflows. Together with mq-hal and repo-signal, it becomes a local assistant stack for safe repository analysis and publish-quality checks. 

mq-mcp + mq-hal + repo-signal
= local AI assistant + safe repo analysis + publish-quality checks

local-first read-only by default repo-aware publish readiness

Project roles

ProjectRole
mq-mcpLocal MCP server, bridge, and safe tool layer.

Safety rules

  1. Default to read-only tools.
  2. Keep repo access scoped through registered local repositories.
  3. Do not run destructive commands without explicit approval.
  4. Do not commit secrets, tokens, private paths, or .env files.
  5. Document write-capable tools in docs/TOOL_SAFETY.md.
  6. Validate documentation whenever a new MCP tool is added.
