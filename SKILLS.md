tool surface, semantic memory, integration docs and release readiness.

| [mcp-tool-safety-maintainer](skills/mcp-tool-safety-maintainer/SKILL.md) | Maintain MCP tool safety metadata and boundaries |
| [release-readiness](skills/release-readiness/SKILL.md) | Prepare and verify releases |

| [semantic-memory-maintainer](skills/semantic-memory-maintainer/SKILL.md) | Maintain semantic memory packs and vector context |
| [terminal-ui-polisher](skills/terminal-ui-polisher/SKILL.md) | Improve terminal and command-surface UX |
